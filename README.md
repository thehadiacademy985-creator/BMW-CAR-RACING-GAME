# BMW CAR RACING GAME - Official Website

A modern, responsive gaming website for the BMW CAR RACING GAME featuring high-speed 3D racing in New York City.

![Website Preview](https://img.shields.io/badge/Status-Active-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)
![Version](https://img.shields.io/badge/Version-1.0.0-orange)

## 🎮 Features

### 🏠 Homepage
- **Bold Hero Section**: Animated 3D racing banner with Canvas-based animation
- **Dynamic Background**: Simulated NYC street racing with animated cars
- **Call-to-Action Buttons**: "Play Now" and "Download Game" with hover effects
- **Statistics Display**: Key game metrics (3 Laps, 4 Cars, NYC Track)
- **Scroll Indicator**: Smooth scrolling guide for users

### 📖 About Section
- Comprehensive game description
- Key features list with checkmarks
- Beautiful gradient text and animations
- Placeholder for game screenshot

### 🚗 Cars Section
- **4 Featured Cars**:
  - BMW M4
  - BMW M5
  - Lamborghini
  - Porsche 911
- Individual car cards with:
  - Custom color-coded placeholders
  - Top Speed specifications
  - Acceleration ratings
  - Handling scores
  - "Select Car" buttons with interactive feedback

### ⚙️ Game Features
- 6 feature cards with icons
- Grid layout with hover effects
- Highlights including:
  - 3D Realistic Graphics
  - NYC Racing Tracks
  - Multiple Cars Selection
  - Smooth Controls
  - High-Speed Competition
  - 3-Lap Racing System

### 🖼️ Gallery Section
- 6-item image gallery
- Responsive grid layout
- Smooth hover animations
- Placeholder images with icons

### 📥 Download Section
- Platform-specific download buttons
  - Windows PC (1.2 GB)
  - Android (850 MB)
- Large, accessible buttons
- Download size information

### 📞 Contact Section
- Contact information display
- Interactive contact form
- Email, location, and phone details
- Form validation and submission handling

### 🔗 Footer
- Multiple footer sections
- Social media links
- Quick navigation
- Support and legal links
- Responsive footer grid
